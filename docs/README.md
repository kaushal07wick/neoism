# Neoism Docs

This directory contains the Docusaurus site for Neoism documentation.

## Install

```bash
npm install
```

## Local Development

```bash
npm start
```

## Build

```bash
npm run build
```

## Writing Guidelines

- Write for Neoism as a terminal-first Neovim IDE with a native desktop app, shared frontend state, daemon-backed web surface, and local agent runtime.
- Do not restore inherited marketing pages, blog posts, or package/install instructions.
- Verify commands, settings, and architecture claims against the current repository.
- Keep desktop, web, daemon, protocol, and shared UI responsibilities scoped clearly.
- Keep generated files and scratch content out of git.
