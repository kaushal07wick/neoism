# Changelog

The public changelog is being reset for Neoism-specific releases.

Historical terminal-engine work still exists in git history, but inherited release notes are no longer published as the Neoism docs changelog.

## Current Development Focus

- Terminal-first Neovim IDE workflows.
- Rust-owned workspace chrome.
- Shared frontend state for desktop and web surfaces.
- Daemon-backed workspace access through `neoism-workspace-daemon` and `neoism-protocol`.
- Local agent server and UI integration.
- Settings/configuration cleanup.
- Documentation rewritten around Neoism as its own product.
