# Reusable Art 16th Century Illuminated Letters

Downloaded from Reusable Art's 16th Century Illuminated Letters collection:

`https://www.reusableart.com/alphabets_letters/full-alphabets/illuminated-letters`

The collection page describes the images as fully public domain and ready for downloading. The local `manifest.json` retains the source page and source image URL for each downloaded letter.

This historical pack is intentionally partial; renderer integrations should fall back to a font-backed style when a requested letter image is missing.
