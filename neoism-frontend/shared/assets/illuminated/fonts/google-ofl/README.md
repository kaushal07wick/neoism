# Google OFL Illuminated Initial Fonts

Downloaded from the Google Fonts repository for Neoism's opt-in illuminated markdown rendering.

- `UnifrakturCook-Bold.ttf`: `https://github.com/google/fonts/tree/main/ofl/unifrakturcook`
- `UnifrakturMaguntia-Book.ttf`: `https://github.com/google/fonts/tree/main/ofl/unifrakturmaguntia`
- `CinzelDecorative-Regular.ttf`, `CinzelDecorative-Bold.ttf`, `CinzelDecorative-Black.ttf`: `https://github.com/google/fonts/tree/main/ofl/cinzeldecorative`
- `PirataOne-Regular.ttf`: `https://github.com/google/fonts/tree/main/ofl/pirataone`
- `MedievalSharp.ttf`: `https://github.com/google/fonts/tree/main/ofl/medievalsharp`
- License: SIL Open Font License 1.1, included as `OFL.txt`.

These are bundled as starter style-pack fonts. User-installable font and SVG/image packs should follow the same manifest model under `shared/assets/illuminated/manifest.json`.
